About Me

I have more than 8 years of professional experience with various languages and programming technologies. My main goal when I write software is to provide efficient, reusable, reliable and well optimized code within an Object-Oriented Design. I prefer to work on Linux because it provides more flexible and reliable working environment.

I'm interested in freelance projects for mobile devices (Apache Cordova), web applications (JavaScript/Node.js), visualizations on HTML5-canvas, audio processing, machine learning, front end applications, one page applications, complex backend processes, etc. My favorite software technologies and languages are JavaScript/Node.js, JavaScript/Apache Cordova, C++.

My passions and motivators are mountain bike riding, hiking, playing Bulgarian traditional flute (Kaval), sound engineering, playing guitar and electric keyboard, music composition, software coding, yoga, swimming and trail running.
